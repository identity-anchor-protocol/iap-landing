5-Minute Drift Demo

This is an experimental local demo. It is useful for understanding verification failure after deliberate state tampering, but it is not the primary user-facing beta workflow.

  1. Start from a clean virtual environment.
  2. Install the current iap-agent package.
  3. Run the local demo script that prepares a baseline state chain.
  4. Verify the baseline state.
  5. Mutate the local state history in the demo.
  6. Verify that the tampered chain fails.
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
python -m pip install iap-agent==0.1.8
python /path/to/iap-sdk/examples/state-drift-demo/demo.py

Expected terminal result includes:

verify_before_ok=True
verify_after_ok=False

The first check confirms the baseline before mutation. The second check fails after deliberate tampering in the demo, showing what a broken verification path looks like.

Use the normal quickstart and user manual for the supported beta flow.
