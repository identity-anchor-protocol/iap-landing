Why Memory Compaction Can Cause Drift

Compaction rewrites context structure. The same facts can produce different prompts after compaction.

This can change downstream behavior without obvious visible changes to users.

In the current beta, IAP does not directly analyze compaction. It can still make those changes easier to inspect if the affected files are part of the tracked local state that gets appended into AMCS and later certified.

Treat this as a technical reasoning note, not as a separate beta product feature.
